# RaumDesign Rudek — Webcreator Redesign

Astro 5 + Tailwind v4 redesign of https://www.raumdesign-rudek.de/ using locally downloaded original website images.

## Commands

```bash
npm install
npm run dev
npm run build
npm run preview
```

## Content source

Original HTML and asset manifest are stored under `original-site/`. All downloaded image assets are copied to `public/images/` and used locally in the redesign.
